Steps to Install Docker on Ubuntu

1. Update your system packages

sudo apt-get update
sudo apt-get upgrade -y

2. Remove any older versions of Docker (if any)

sudo apt-get remove docker docker-engine docker.io containerd runc

3. Set up the Docker repository

Install required packages:

sudo apt-get install -y ca-certificates curl gnupg lsb-release

Add Docker’s official GPG key:

sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

Set up the stable Docker repository:

echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

4. Update the package index

sudo apt-get update

5. Install Docker components

sudo apt-get install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

6. Verify Docker installation

sudo docker --version

7. Start and enable Docker

sudo systemctl start docker
sudo systemctl enable docker

8. (Optional) Run Docker as a non-root user

Add your user to the docker group:

udo usermod -aG docker jenkins
        sudo systemctl restart jenkins

Log out and back in to apply the changes.

Troubleshooting

  • If your system’s architecture or Ubuntu version is not supported by Docker's repositories, you might need to use an older Docker version or a manual installation method.
  • Ensure that you are using a supported Ubuntu version (e.g., Ubuntu 20.04, 22.04).
